已实现的函数

aes_encrypt

  1. 第0轮只做AddRoundKey，即matrix与key异或（加法）
  2. 随后进入第1~key_rounds轮循环，每次执行
    1. 把matrix中的16字节替换成sbox中的值
    2. matrix行转列
    3. matrix循环左移
    4. 做MixColumn（如果是最后一轮不做乘法）
    5. 做AddRoundKey，即matrix与key异或（加法）

注. 2.2 + 2.3步（分割线上方、下方为不同的实现过程，最终采用上方的实现方式，即先行转列，再ShiftRow） ↩

MixColumn

  1. 对p指向的$4\times4$矩阵$m$中的4列（注意，是列！）做乘法运算;

  2. 这里的乘法是指有限域$GF(2^8)$多项式模$(X^4+1)$乘法,具体步骤请参考教材p.61及p.62;

  3. AES加密时采用的被乘数a为多项式$3*X^3 + X^2 + X + 2$, 用数组表示为

  4. AES解密时采用的被乘数a为加密所用多项式的逆, 即$\{0x0B, 0x0D, 0x09, 0x0E\}$;

  5. 矩阵$m$中的4列按以下顺序分别与a做乘法运算:

    • 第0列: 由m[0][0], m[1][0], m[2][0], m[3][0]构成
    • 第1列: 由m[0][1], m[1][1], m[2][1], m[3][1]构成
    • 第2列: 由m[0][2], m[1][2], m[2][2], m[3][2]构成
    • 第3列: 由m[0][3], m[1][3], m[2][3], m[3][3]构成

  6. 乘法所得4列转成4行, 保存到p中, 替换掉p中原有的矩阵;

  7. do_mul用来控制是否要做乘法运算, 加密最后一轮及解密第一轮do_mul = 0;

MixColumnInverse

  1. 把p指向的$4\times4$矩阵$m$中的4行（与之前的MixColumn不同，是行！）逐行与矩阵a做乘法运算;

  2. 这里的乘法是指有限域$GF(2^8)$多项式模$(X^4+1)$乘法;

  3. 被乘数a为多项式$0xB X^3 + 0xD X^2 + 0x9 * X + 0xE$的4个系数, 即$\{0x0B, 0x0D, 0x09, 0x0E\}$, 展开为矩阵就是:

  4. 矩阵$m$中的4行逐行与a做乘法运算后, 按以下顺序转成矩阵$t$中的列:

    • 第0列: $a * m[0]$
    • 第1列: $a * m[1]$
    • 第2列: $a * m[2]$
    • 第3列: $a * m[3]$

  5. 复制$t$到p中, 替换掉p中原有的矩阵;

  6. do_mul用来控制是否要做乘法运算, 加密最后一轮及解密第一轮do_mul = 0;
